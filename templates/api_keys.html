How to use the API

Example request with curl:

curl -X POST {{ request.host_url }}api/v1/scan \
  -H "X-API-Key: <your-key>" \
  -H "Content-Type: application/json" \
  -d '{"target": "example.com"}'

Always send the key in the X-API-Key header, never as a query parameter.
