Come usare l'API

Esempio di chiamata con curl:

curl -X POST {{ request.host_url }}api/v1/scan \
  -H "X-API-Key: <la-tua-chiave>" \
  -H "Content-Type: application/json" \
  -d '{"target": "example.com"}'

La chiave va sempre inviata nell'header X-API-Key, mai come query parameter.
